# ❓Adivinhe o Número
![Python](https://img.shields.io/badge/Python-3776AB.svg?style=for-the-badge&logo=Python&logoColor=white)

Projeto iniciante de estudo para interfaces gráficas em Python

---

### 📑 Sobre o Projeto

Você deve adivinhar um número aleatório sorteado pelo sistema. A cada palpite, o sistema informará se o número correto é maior ou menor que o valor escolhido. Você terá até 10 tentativas para acertar, independentemente da dificuldade. O jogo tem 3 níveis de dificuldade, mudando o intervalo que o número pode ser sorteado:

 - 🟢 Fácil: 1 - 25
 - 🟡 Médio: 1 - 60
 - 🔴 Difícil: 1 - 100

---

### 🌐 Visite meu projeto

Baixe a pasta desse reposittório do github e execute o arquivo ```Jogo.exe```

---

### 🛠️ Tecnologias Utilizadas

Para garantir uma interface leve, rápida e elegante, utilizei:

* **KIVY:** Estruturação de Telas e Interface Gráfica
* **PYTHON:** Desenvolvimento da lógica de programção do jogo.
